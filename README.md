# Planetarium VR

Projeto de realidade virtual desenvolvido na Unity com foco na criação de um museu interativo do Sistema Solar.

## Sobre
O **Planetarium VR** é um projeto acadêmico que busca recriar um ambiente espacial imersivo, inspirado em planetários e museus interativos. O cenário conta com fundo estrelado, corpos celestes e proposta educativa voltada à exploração do Sistema Solar.

## Tecnologias utilizadas
- Unity
- C#
- Universal Render Pipeline (URP)

## Funcionalidades
- Ambiente espacial com fundo estrelado
- Planetas em rotação
- Interação com elementos da cena
- Proposta educativa e imersiva

## Status do projeto
Em desenvolvimento.
